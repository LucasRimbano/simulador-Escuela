# 🎓 Simulador de Inscripción Escolar

Aplicación web desarrollada en **HTML, Bootstrap y JavaScript** que permite simular el proceso de inscripción de alumnos, gestionar sus datos y visualizar la información de manera dinámica.

---

## 📌 Descripción

Este proyecto representa un sistema básico de gestión escolar donde el usuario puede:

* Ingresar datos de alumnos mediante un formulario
* Validar la información ingresada
* Almacenar los datos en memoria y en el navegador (LocalStorage)
* Visualizar una lista de alumnos registrados
* Gestionar la información mediante diferentes acciones

El objetivo principal es simular un flujo real de inscripción escolar y aplicar conceptos fundamentales de JavaScript.

---

## ⚙️ Funcionalidades principales

✔️ Registro de alumnos con:

* Nombre
* Apellido
* Edad
* Sexo
* Año lectivo
* Orientación (Bachiller)

✔️ Validaciones de formulario:

* Control de datos numéricos
* Verificación de campos obligatorios
* Manejo de errores visuales

✔️ Gestión de datos:

* Visualizar lista de alumnos
* Eliminar todos los registros
* Guardar datos en LocalStorage
* Cargar datos previamente guardados

✔️ Alertas interactivas:

* Uso de **SweetAlert2** para notificaciones de éxito, error y confirmación

---

## 🧠 Tecnologías utilizadas

* HTML5
* CSS3 (Bootstrap 5)
* JavaScript (DOM, eventos, arrays, objetos)
* LocalStorage
* SweetAlert2

---

## 🎯 Objetivo del proyecto

Este simulador fue desarrollado como parte de una práctica para consolidar conocimientos en:

* Manipulación del DOM
* Programación con JavaScript
* Validación de formularios
* Persistencia de datos en el navegador
* Diseño de interfaces con Bootstrap

---

## 🚀 Cómo usar el proyecto

1. Completar el formulario con los datos del alumno
2. Presionar el botón **"Enviar"** para registrar
3. Utilizar los botones:

   * **Ver Lista** → muestra los alumnos cargados
   * **Guardar Alumnos** → guarda los datos en el navegador
   * **Cargar alumnos guardados** → recupera los datos almacenados
   * **Limpiar Lista** → elimina todos los registros
